 Student Skills Tracker
A C# console application for tracking students' skills and progress in training programs. Built with .NET 8 and SQLite for persistent data storage.

 Student Management
Add, update, and delete student records
View detailed student profiles with skill progress
Search students by name or email
Archive deleted students for record keeping

 Skills Tracking
Define skills with categories (Programming, Web Development, Soft Skills, etc.)
Set passing thresholds for each skill
Track individual student progress on each skill
Visual progress bars for quick status assessment

 Training Programs
Create training programs with required skills
Define completion thresholds for certification
Check student readiness for program certification
View which skills students are missing for completion

 Analytics & Reporting
Individual student progress reports
Class-wide statistics and averages
Identify skills needing attention across all students
Certification readiness assessment

 Tech Stack
Framework: .NET 8
Language: C# 
Database: SQLite (via Microsoft.Data.Sqlite)
Architecture: Repository Pattern with Separation of Concerns
